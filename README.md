# [爱逗B](http://www.idleb.me)
> 纪念一下逗B的几年

## 项目简介
基于SpringBoot 2.x、Vue 2.x前后端分离的开源Blog系统，idle说明失业中，B指Blog。

## 使用技术
### 后端
- JDK8
- SpringBoot 2.x
- Mybaits(Mybaits-Plus) 不要问为什么不用JPA,因为我喜欢自在的感觉
- MySQL 5.7
- Spring Security
- Redis
- RabbitMQ
- ElasticSearch
- Lombox
- Swagger
- Druid
- Jasypt

### 前端
- Node.js 8.0+
- Vue 2.x

### 工具
- IDEA
- VS Code
- Git
